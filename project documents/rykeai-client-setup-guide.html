What You Need

Four Services, One Platform

RYKE AI relies on four third-party services. You will create a free account for each, navigate to the relevant settings page, and copy the API keys listed below.

💬
Step 1
Twilio

Sends and receives SMS and MMS messages. Powers all coaching conversations for Android and non-iPhone users.

🤖
Step 2
Anthropic (Claude API)

The AI brain behind all coaching conversations, food photo analysis, and crisis detection.

🍎
Step 3
SendBlue

Delivers coaching messages as iMessages (blue bubbles) to iPhone users with automatic SMS fallback.

💳
Step 4
Stripe

Handles payment card collection, free trial activation, and monthly subscription billing.

Step 1 of 4
📱
Twilio
https://www.twilio.com/try-twilio
SMS/MMS messaging — sends and receives all coaching text messages
Starting cost
Free trial credit
1
Create a Twilio account

Go to the link below and sign up with your email. Twilio gives you free trial credit to get started — no credit card required initially.

https://www.twilio.com/try-twilio
ℹ️

Verify your email address when prompted. Twilio will also ask you to verify a phone number — use a real mobile number you have access to.

2
Get a phone number

After signing in, go to Phone Numbers → Manage → Buy a number. Search for a number with SMS and MMS capabilities. With the free trial you can get one number at no charge.

📌

Select a number that shows both SMS ✓ and MMS ✓ capability icons.

3
Copy your credentials

Go to Account → API keys & tokens (or your Console dashboard). Copy the three values below.

TWILIO_ACCOUNT_SIDStarts with AC — found on your Console dashboardRequired
TWILIO_AUTH_TOKENClick "show" next to Auth Token on the dashboardRequired
TWILIO_PHONE_NUMBERThe number you purchased — in E.164 format e.g. +15551234567Required
⚠️

Keep your Auth Token secret. Never share it or post it publicly. Treat it like a password.

4
Configure the webhook URL

Go to Phone Numbers → Manage → Active numbers → click your number → Messaging Configuration. Under "A message comes in" set the webhook URL to your backend URL. Leave this blank for now — your developer will fill it in after deployment.

📌

Webhook URL format: https://api.ryke.ai/v1/webhooks/sms — your developer will provide the exact URL.

Step 2 of 4
🤖
Anthropic (Claude API)
https://console.anthropic.com
AI coaching engine — powers all conversations, food analysis, and safety detection
Starting cost
Pay as you go
1
Create an Anthropic account

Go to the Anthropic Console and sign up. Use a business email address.

https://console.anthropic.com
2
Add billing (required to use the API)

Go to Settings → Billing and add a credit card. Add an initial credit of at least $10 to get started. You will only be charged for what you use — Claude Haiku (the model RYKE AI uses) costs roughly $0.25 per 1,000 messages.

✅

At 100 active users sending 10 messages/day, expect roughly $7–$10/month in Claude costs. This is extremely low compared to Twilio SMS costs.

3
Create an API key

Go to API Keys in the left sidebar and click "Create Key". Give it a name like "RYKE AI Production". Copy the key immediately — it is only shown once.

ANTHROPIC_API_KEYStarts with sk-ant- — shown only once at creationRequired
⚠️

Copy and save this key immediately in a secure location (password manager). Anthropic does not show it again — if lost, you must create a new one.

Step 3 of 4
🍎
SendBlue
https://sendblue.co
iMessage delivery — sends coaching as blue bubbles to iPhone users
Starting cost
Free tier available
1
Create a SendBlue account

Go to SendBlue and sign up. SendBlue provides iMessage-as-a-service — your iPhone users will receive coaching as native iMessages.

https://sendblue.co
ℹ️

iPhone users will see RYKE AI messages as blue bubbles. Android users and basic phone users automatically fall back to regular SMS through Twilio — no action needed from you.

2
Get your API credentials

After signing in, go to Dashboard → API Keys (or Developer settings). Copy both values below.

SENDBLUE_API_KEY_IDYour public API Key ID from the dashboardRequired
SENDBLUE_API_SECRET_KEYYour API Secret Key — keep this privateRequired
3
Note your sending number

Note the phone number assigned to your SendBlue account. This is the number iPhone users will receive messages from.

SENDBLUE_SENDING_NUMBERPhone number in E.164 format e.g. +15551234567Optional
Step 4 of 4
💳
Stripe
https://dashboard.stripe.com/register
Payment processing — collects cards, activates free trials, handles monthly billing
Fee per transaction
2.9% + $0.30
1
Create a Stripe account

Go to Stripe and register. Use your business email. Stripe will ask for basic business details to activate live payments — you can start in test mode immediately without completing this.

https://dashboard.stripe.com/register
2
Create your subscription product and pricing

Go to Product Catalogue → Add Product. Create a product called "RYKE AI Individual" with a recurring price of $20.00/month. After saving, copy the Price ID (starts with price_).

STRIPE_PRICE_ID_INDIVIDUALThe price ID for the $20/mo plan — starts with price_Required
📌

On the product page, click the price to expand it. The Price ID appears under the price details — it looks like price_1ABC123XYZ.

3
Get your API keys

Go to Developers → API Keys. You will see two keys — use the test keys first (for development) and switch to live keys when you go live.

STRIPE_SECRET_KEYSecret key — starts with sk_test_ (test) or sk_live_ (production)Required
⚠️

Only share the secret key with your developer. Never expose it in a browser, public repo, or email.

4
Set up a webhook and get the webhook secret

Go to Developers → Webhooks → Add endpoint. Your developer will provide the exact URL after deployment. For now, add a placeholder and select these events to listen to:

✓ customer.subscription.created    ✓ customer.subscription.updated    ✓ customer.subscription.trial_will_end    ✓ invoice.payment_failed    ✓ invoice.payment_succeeded    ✓ customer.subscription.deleted

After creating the webhook, click "Reveal signing secret" and copy it.

STRIPE_WEBHOOK_SECRETWebhook signing secret — starts with whsec_Required

Stores all API keys securely for the CI/CD pipeline and deployments

Open the GitHub Secrets page

Go to your RYKE AI repository on GitHub, then navigate to Settings → Secrets and variables → Actions.

https://github.com/avancerasolution/ryke-ai/settings/secrets/actions
2
Add each secret below

Click "New repository secret" for each row in the table. Enter the exact name shown and paste the value you collected from each service above.

Secret NameWhere to get itRequired
ANTHROPIC_API_KEYAnthropic Console → API KeysRequired
TWILIO_ACCOUNT_SIDTwilio Console → DashboardRequired
TWILIO_AUTH_TOKENTwilio Console → Dashboard → ShowRequired
TWILIO_PHONE_NUMBERTwilio → Phone Numbers → Active NumbersRequired
SENDBLUE_API_KEY_IDSendBlue Dashboard → API KeysRequired
SENDBLUE_API_SECRET_KEYSendBlue Dashboard → API KeysRequired
